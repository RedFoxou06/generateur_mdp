## Générateur de mot de passe | Projet personnel
**Description :**  
M'ayant été interessé à la cybersécurité avec une attaque brut force sur un mot de passe simple, je me suis aussi interessé à différentes façons de s'en protéger. L'une étant d'avoir un bon mot de passe.
J'ai créé une application qui générait un mot de passe selon certains critères que l'on peut sélectionnés.
A voir pour le rajouter ou le mettre sur une page HTML grâce à du Javascript pour qu'il soit plus simple d'utilisation.

**Langages utilisés :**
- Python

**Logiciels utilisés :**
- IntelliJ
- PyInstaller
- Git
